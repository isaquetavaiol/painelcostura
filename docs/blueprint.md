# **App Name**: Split It!

## Core Features:

- Bill Input: Enter the total bill amount.
- Tipper Info: Enter the percent you would like to tip the server.
- Split Number: Enter the number of people splitting the bill.
- Calculate tip: Based on provided information, calculate how much to tip total, and the amount for each person.
- Total output: Shows the subtotal, tip and grand total for bill, and also the totals split by the number of people.

## Style Guidelines:

- Primary color: Deep pink (#D14D72) for a fun, energetic vibe.
- Background color: Light pink (#F9E4E9), almost white, provides a soft backdrop.
- Accent color: Purple (#A459D1) as an accent color.
- Body and headline font: 'Poppins', a geometric sans-serif for short bursts of contemporary precision.
- Use simple, outline-style icons related to money and dining.
- Use subtle animations when calculations are performed.
