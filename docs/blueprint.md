# **App Name**: Atelier Insights

## Core Features:

- Service Profit Tracking: Visualize profits across tailoring adjustments, dresses, and customizations.
- Comparative Analytics: Calculate month-over-month revenue growth percentages with visual graphs.
- Expense Breakdown: Categorize and display expense distribution across materials, rent, and supplies.
- Automated Alert System: Highlight overdue payments and unmet targets using a visual alert system.
- Daily Financial Summary: Provide a clear summary of daily revenue, expenses, and net profit.
- Price Simulator: Suggest optimal project prices using material costs and labor time.
- Revenue Forecasting: Estimate future income based on current project statuses.

## Style Guidelines:

- Primary color: #2E303F
- Secondary color: #63938C
- Background color: #EBE5D0
- Headline font: 'Playfair' serif font for an elegant, high-end feel. Body font: 'PT Sans' sans-serif font for readability.
- Crisp, clear icons representing various aspects of the atelier, such as sewing machines, fabric, money, charts, and alerts.
- A clean and structured layout that allows for easy navigation between financial summaries, analytical reports, and customer segmentation tools.
- Subtle transitions and loading animations to provide feedback without overwhelming the user.
